# Pong_game_in_Rust
Я зробив гру Pong на Rust.
## Що таке Pong?
Pong — це одна з перших успішних аркадних відеоігор, симулятор настільного тенісу, розроблений компанією Atari у 1972 році, яка започаткувала індустрію відеоігор, а також назва серії домашніх ігрових приставок, випущених пізніше.
Гра має надзвичайно простий геймплей: два гравці відбивають м'ячик прямокутними "ракетками" на екрані, намагаючись забити гол супернику. 
## Що таке Rust?
Rust — це сучасна багатопарадигмова мова програмування, що фокусується на швидкості, безпеці пам'яті та паралелізмі, розроблена Mozilla Research як системна мова з продуктивністю на рівні C++, але без багатьох її проблем, що робить її ідеальною для ОС, веб-серверів, вбудованих систем, ігор, блокчейну та інших високопродуктивних застосунків.
